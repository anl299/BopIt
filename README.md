Instructions for Github Assignment

2. Create a branch
• Each team member should join the repository
• Create a new branch named firstnameLastInitial-edit
• Do not make changes directly to the main branch
3. Make a change
• Before editing, confirm that your new branch is selected in the branch dropdown
• Each team member should edit the source file by adding, at minimum, a comment
containing their name. Feel free to include other changes.
4. Commit
• Each team member should then commit the change that they made
5. Submit a Pull Request
• Everyone should open a Pull Request from your own branch
• Request to merge into the main branch
6. Merge Changes
• Wait until all team members have submitted pull requests
• The project maintainer should review and merge each pull request
7. Clone the repository
• After all pull requests have been merged, each team member should clone the
repository, creating a local copy of the project on your computer
• Verify that all teammate changes are present
